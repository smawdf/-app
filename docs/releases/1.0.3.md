# 高糖小食 1.0.3

本次更新修复不同拍摄比例的图片在固定容器中出现空白的问题。

## 更新内容

- 店铺封面、头像、菜品缩略图、订单图片和甜蜜时刻缩略图改为居中铺满。
- 手机竖拍或横拍照片不再在方形、圆形和横向容器中出现明显空白。
- 点菜详情和发现详情保留完整原图展示，不拉伸图片。
- 甜蜜时刻点击缩略图后，可在设置弹窗查看完整照片。
- 上传图片仍按原始比例压缩，云端图片内容不会被裁切保存。

## 安装

下载 `1.0.3.apk`，在 Android 7.0 或更高版本设备上安装。同签名覆盖安装会保留本地数据。

## 云端要求

本版本不需要执行新的 Supabase SQL。
