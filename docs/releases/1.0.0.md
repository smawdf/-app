# 高糖小食 1.0.0

高糖小食第一版私用正式发布。这是一款只服务一对情侣的 Android 点菜 App，不是多商家外卖平台。

## 主要功能

- 登录、注册、头像和昵称设置。
- 情侣邀请码绑定、角色分工、绑定通知和解除绑定。
- 饲养员维护店铺、分类、菜品、库存、封面和公告。
- 吃货点菜、加入购物车、结算和提交订单。
- 饲养员确认接单并完成订单。
- 发现页搜索中文菜品并加入我的店铺。
- 糖糖币余额、充值和记录。
- 纪念日和甜蜜时刻，支持菜品图片或相册、拍照上传共同照片。
- 个人资料、情侣关系、店铺、菜单、订单和偏好设置云端同步。

## 安装

下载 `高糖小食-1.0.0.apk`，在 Android 7.0 或更高版本设备上安装。覆盖安装会保留同签名旧版本的本地数据。

## 云端要求

Supabase SQL 需要按 `table/README.md` 的顺序执行至 `37_sweet_moment_images.sql`。
