# 高糖小食 1.0.5

本次更新继续完善店铺分类的一致性，确保菜品编辑与发现页只使用当前实际分类。

## 更新内容

- 菜品详情和编辑弹窗不再固定显示“主食、小吃、饮品”等快捷分类。
- 快捷分类现在只展示店铺当前实际存在的分类。
- 发现页加入菜品时，会验证搜索结果分类是否属于当前店铺。
- 搜索结果分类不存在时，自动选择当前店铺的第一个分类。
- 店铺没有任何分类时，菜品统一进入“未分类”。

## 安装

下载 `1.0.5.apk`，在 Android 7.0 或更高版本设备上安装。同签名覆盖安装会保留本地数据。

## 云端要求

本版本不需要执行新的 Supabase SQL。
