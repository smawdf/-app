# 高糖小食 1.0.4

本次更新统一了店铺分类管理逻辑，并修复发现页加入店铺时出现已删除分类的问题。

## 更新内容

- “招牌”和“热销”改为普通分类，可像主食、饮品等分类一样创建、重命名和删除。
- 移除自动生成的热销分类，以及菜品编辑中的招牌专属开关。
- 点菜页严格按照菜品当前分类展示，不再把全部菜品自动归入热销。
- 发现页加入店铺时只读取店铺当前实际分类，不再显示已删除的历史分类。
- 删除分类后，原分类中的菜品会迁移到可用分类，避免菜品丢失。

## 安装

下载 `1.0.4.apk`，在 Android 7.0 或更高版本设备上安装。同签名覆盖安装会保留本地数据。

## 云端要求

本版本不需要执行新的 Supabase SQL。
