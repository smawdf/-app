# 高糖小食 1.0.1

本次更新集中改善不同尺寸设备上的显示效果，并完善解绑历史、订单信息和通知体验。

## 更新内容

- 优化小屏、大字体、横屏手机和平板布局，减少文字遮挡和无效空白。
- 修复订单列表与购物车在部分情况下无法滑动的问题。
- 优化情侣昵称、菜品名称、推荐卡片和页面标题的换行显示。
- 首页订单通知根据内容自动调整高度，减少卡片底部多余空白。
- 完善输入弹窗的滚动、窗口高度和软键盘避让。
- 解绑后双方仍可分别查看已有共同订单、店铺、菜单和纪念日。
- 订单页面完善店铺封面、点单人资料和历史订单只读状态。
- 支持从通知进入对应订单，完善订单进度通知。
- 修复发现页每日推荐读取，缺图继续使用下厨房与 Bing 兜底。

## 安装

下载 `1.0.1.apk`，在 Android 7.0 或更高版本设备上安装。同签名覆盖安装会保留本地数据。

## 云端要求

Supabase SQL 需要按 `table/README.md` 的顺序执行至 `38_unpair_personal_history.sql`。
