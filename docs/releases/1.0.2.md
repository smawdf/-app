# 高糖小食 1.0.2

本次更新完善图片显示和应用内在线更新，继续保持私用版轻量流程。

## 更新内容

- 图片上传会压缩文件大小，并按原始比例缩放，不裁切图片内容。
- 菜品图、店铺封面、头像和甜蜜时刻完整显示，图片比例不匹配时保留少量留白。
- “我的 -> 版本与更新”支持检查 GitHub 最新正式版本。
- 支持查看 Release 更新日志、下载 APK、显示下载进度并调起系统安装器。
- 首次安装更新时，Android 可能要求允许高糖小食安装未知应用。

## 安装

下载 `1.0.2.apk`，在 Android 7.0 或更高版本设备上安装。同签名覆盖安装会保留本地数据。

## 云端要求

Supabase SQL 需要按 `table/README.md` 的顺序执行至 `38_unpair_personal_history.sql`。
