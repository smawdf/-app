# 高糖小食更新日志

## 1.0.3 (36) - 2026-07-16

图片显示策略修复。

- 固定尺寸的封面、头像和列表缩略图改为居中铺满，消除手机竖拍、横拍图片产生的空白。
- 点菜详情和发现详情继续完整展示原图，不拉伸图片。
- 甜蜜时刻列表图片铺满缩略图，点击后可在设置弹窗查看完整照片。
- 图片上传继续按原始比例压缩，只调整展示方式，不破坏云端原图内容。

## 1.0.2 (35) - 2026-07-16

图片体验与版本更新能力优化。

- 图片上传继续压缩占用空间，保持原始比例，不裁切内容。
- 菜品、店铺封面、头像和甜蜜时刻改为完整比例展示，避免图片边缘被裁掉。
- 在“我的 -> 版本与更新”中检查 GitHub 最新 Release。
- 支持显示远程更新日志、下载进度和系统安装确认。
- 适配 Android 安装未知应用权限提示，保持同签名覆盖安装和本地数据。

## 1.0.1 (34) - 2026-07-15

体验优化与稳定性更新。

- 优化小屏、大字体、横屏手机和平板的响应式布局。
- 修复订单列表和购物车内容较少但超出屏幕时无法滑动的问题。
- 修复顶部标题、情侣昵称、菜品名称和推荐卡片的文字遮挡与截断。
- 首页订单通知改为内容自适应高度，减少普通字体下的多余空白。
- 完善地址、个人资料、情侣绑定、店铺分类和纪念日编辑弹窗的键盘避让。
- 扩大纪念日日历、甜蜜时刻图片编辑等控件的触控区域。
- 解绑后双方仍可分别查看已有共同订单、店铺、菜单和纪念日信息。
- 订单列表和详情完善店铺封面、点单人资料与历史订单只读状态。
- 通知支持点击进入对应订单，并完善新版本 Android 的订单进度通知。
- 修复发现页每日推荐读取问题，继续使用下厨房与 Bing 图片兜底链路。
- 优化云端同步、订单一致性和本地数据库迁移验证。

## 发布说明

- 这是情侣两人使用的私有应用，不面向应用商店和多商家外卖场景。
- 云端数据库需要按 `table/README.md` 的顺序执行至 `37_sweet_moment_images.sql`。
- Release APK 使用项目本地签名配置生成，签名文件和密钥信息不提交到仓库。
